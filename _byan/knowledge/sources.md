## Regles d'utilisation

1. BYAN ne cite que les sources de ce fichier dans un `[CLAIM]`
2. Toute source non listee ici = `[REASONING]` ou `[HYPOTHESIS]`
3. Pour ajouter une source : verifier l'URL manuellement, ajouter avec niveau et date
4. Format : `**Nom** : URL`

---

## Ajouter une source

Pour enrichir cette base, l'utilisateur peut coller une URL en session.
BYAN extrait les claims, l'utilisateur valide, la source est ajoutee avec :
- Niveau de preuve (LEVEL-1 a LEVEL-4)
- Date de verification
- Domaine concerne
