## Couche Vivante vs Immuable

```
+------------------------------------------+
|             NOYAU IMMUABLE               |
|  - 3 verites du createur                 |
|  - Ne change JAMAIS                      |
|  - Identique en essence pour tous        |
|  - Adapte en formulation par agent       |
+------------------------------------------+
|           COUCHE VIVANTE                 |
|  - Evolue par session                    |
|  - Validee par utilisateur               |
|  - Anti-dissonance check (vs noyau)      |
|  - Revision periodique (14 jours)        |
|  - Max 2 entries/session                 |
+------------------------------------------+
|           SOUL MEMORY                    |
|  - Journal chronologique                 |
|  - Types: INSIGHT, TENSION, RESONANCE,   |
|    REVISION                              |
|  - Declencheurs: exit hook, mid-session  |
|  - Historique consultable                |
+------------------------------------------+
```
